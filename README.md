# smart-grid-openenv

## 🚀 Quick Setup for Friends & Collaborators

To get esattamente the same setup as me, including all RL libraries and OpenEnv tools:

1. **Clone the repository.**
2. **Setup the Virtual Environment:**
   ```bash
   python -m venv venv
   .\venv\Scripts\activate   # On Windows
   source venv/bin/activate  # On macOS/Linux
   ```
3. **Install Dependencies:**
   ```bash
   pip install -r smart_grid/server/requirements.txt
   ```
4. **Agent Skills & Training:**
   All agent instructions and pre-trained models are already included in the repo! 
   Run `python -m smart_grid.test_env` to confirm it works.
